# 🏥 Hospital Management System

A full-stack **Hospital Management System** built using **Django (Python)** to manage appointments, doctors, and patient data efficiently.

---

## 🚀 Features

- 🧑‍⚕️ Doctor Management
- 📅 Appointment Booking System
- 🧾 Patient Details Handling
- 🖥️ Clean UI with Bootstrap
- 🔐 Admin Panel (Django Admin)
- 📊 Organized Database Models

---

## 🛠️ Tech Stack

- **Backend:** Django, Python  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite  
- **Tools:** Git, GitHub
